### Реалізуйте власний менеджер пам’яті з аналогами malloc() та free().
Спочатку глобально ініціалізуємо великий масив з якого ми будемо брати пам'ть для `malloc`.
Для реалізації аналогу `malloc` створюємо зміну `counter` яка буде в ролі лічильника вказувати на першу вільну адресу,в функцію буде передаватися розмір,якщо сума лічильника і розміра більше за розмір нашого масиву пам'яті,то повертаємо помилку.Після цього ствоюємо вказівник в який передаємо адресу нашого лічильнка(адресу першої ж вільної комірки),зміщуємо лічильник,та повертаємо адресу вільної комірки.Функцію `free` робимо примітивною просто скидаємо значення лічильника до нуля.
```

#include <stdio.h>
char memory[1024];
size_t counter=0;
void* my_malloc(size_t size){
        if(counter+size>1024){
                return NULL;
        }
        void* ptr=&memory[counter];
        counter+=size;
        return ptr;
}
void* my_free(void* ptr){
        counter = 0;
}

int main(){
        int* mal = (int*)my_malloc(sizeof(int));
        *mal = 1000;
        printf("mal = %d\n", *mal);
        my_free(mal);
        return 0;
}
```

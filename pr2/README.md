# Завдання 1
#### Напишіть програму для визначення моменту, коли time_t закінчиться.Дослідіть, які зміни відбуваються в залежності від 32- та 64-бітної архітектури. Дослідіть сегменти виконуваного файлу.
Для того щоб визначити коли закічниться time_t ми створюємо зміну та намагаємось її переповнити,подвоюємо число так щоб кожен з 64 бітів мав значення 1,потім коли зміну вже переповнено ми отримаємо негативне число і виходимо з цикул,віднімаємо 1 від змінної і отримаємо максимально можливе число,після чого програма виведе значення цього числа,і дату.
```
Max time_t: 9223372036854775807
Date: (null)
```
Оскільки число занадто велике(приблизно 300 млрд років),то дата виводиться (null).Для того щоб команда порахуквала для 32-бітної структури,ставлю в Ubuntu multilib
```
sudo apt update
sudo apt install gcc-multilib
```
Компілюємо
```
gcc -m32 main.c -o Ex1_32
```
В результаті отримаємо
```
./Ex1_32
Max time_t: 2147483647
Date: Tue Jan 19 03:14:07 2038
```
За допомогою команди **size** отримаємо інформацію про сегменти
```
 size Ex1
   text    data     bss     dec     hex filename
   1733     616       8    2357     935 Ex1
```
Для 32-бітної:
```
size Ex1_32
   text    data     bss     dec     hex filename
   1595     312       4    1911     777 Ex1_32

```
